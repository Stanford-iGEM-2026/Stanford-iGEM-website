# Stanford iGEM Website

A visually dynamic website for the Stanford University iGEM team, built with React, Vite, HTML, CSS, and JavaScript.

## Brand Colors

- Crimson: `#8e1918`
- Teal: `#1c7170`
- White & Black

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Sections

- **Hero** — Bioengineering animations, parallax DNA helixes, molecular canvas
- **Gold Medals** — Animated medal counter
- **About** — What is Stanford iGEM
- **Impact** — People helped over the years
- **Programs** — Tabbed section including SIBRP
- **Projects** — Three featured project cards
- **Support** — Donate and mentor CTAs
- **Footer** — Decorative wave and DNA strip navigation

## Customization

Update social media links in `src/components/Navbar.jsx` and `src/components/Footer.jsx`.
Replace placeholder project content in `src/components/Projects.jsx`.
Adjust medal count and impact numbers in `GoldMedals.jsx` and `Impact.jsx`.
